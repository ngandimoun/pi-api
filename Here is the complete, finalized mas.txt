Function: Instead of prompting Nano Banana 2 from scratch, this endpoint queries your Supabase vector database for "Golden Reference" aesthetics. It feeds the model a visual anchor (e.g., "Information-Dense Japanese Ad" or "Minimalist Luxury") to guide the generation.










MODULE 2: Core E-Commerce (The Workhorses)

These endpoints automate the graphic design pipeline for Shopify merchants, marketing agencies, and Ad-Tech platforms.

3. The Zero-Prompt Ad Engine (POST /v1/ads/from-url)

Function: Takes a product_url, brand_id, and target_locale. It scrapes the product, cuts it out, generates a culturally localized background, writes the copy, and deterministically composites the perfect ad.

4. The Omni-Channel Reflow API (POST /v1/ads/reflow)  LOCALISED CONTENTS   

Function: Takes an existing generated ad and intelligently rearranges the text, logo, and product to perfectly fit different aspect ratios (e.g., 9:16 for Stories, 16:9 for Web Banners) without blindly cropping.

5. The Async Bulk Catalog Engine (POST /v1/catalogs/batch) instead let do A/B up to 20

Function: Accepts a JSON array of up to 10,000 product URLs and a webhook. Uses Trigger.dev/Mastra to process the entire catalog overnight, turning ugly inventory into premium lifestyle ads at scale.
