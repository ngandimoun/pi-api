Here is the Complete, Uncompromised 2026 Tech Stack for the DreamCut API, including the missing Brand Extraction Engine. This is the exact blueprint to hand to your engineering team.

LAYER 1: The Gateway & Security (The Front Door)

You are a global API. You need to handle traffic from New York and Tokyo with zero latency, while protecting against DDoS attacks.

Edge CDN & WAF: Cloudflare (Routes traffic, caches static assets, blocks bad bots).

API Key Management: Unkey.dev (Generates dc_live_... keys, handles rate limiting in 2ms at the Edge, and syncs directly to Stripe for usage-based metered billing).

LAYER 2: The Core Backend (The Logic)

This routes the requests and structures your codebase.

Framework: NestJS (Strict, modular, TypeScript-first. Perfect for managing 10 different API microservices without turning into spaghetti code).

Async Job Processing: Trigger.dev v3 (Crucial. When a request takes 10 seconds to generate an ad, Vercel will time out. Trigger.dev puts it in a durable background queue and ensures it never fails).

LAYER 3: The Ingestion & Brand Extraction Engine (The Missing Piece!)

How you turn zara.com into a structured JSON "Brand Brain."

AI Web Scraper: Browserbase or Firecrawl (These bypass Cloudflare anti-bot protections. They load the Javascript-heavy Shopify site and take a pristine 4K screenshot of the homepage).

Visual Design Parser: Claude 3.7 Vision (via Vercel AI SDK). You pass the homepage screenshot to Claude Vision with the strict prompt: "Extract the primary hex code, secondary hex code, and typography hierarchy. Return ONLY a JSON object."

Asset Extractor: Cheerio / CSSOM. A lightweight script that scrapes the <head> of the website to find the exact .otf/.woff2 font files and the .svg logo URL to save into your database.

LAYER 5: The Visual Rendering Pipeline (The Factory)

Where the actual pixels are made. No hallucinations allowed.

Serverless GPU Compute: Fal.ai (You send the prompt here to generate the lifestyle background using Nano Banana 2 or Flux. You pay per millisecond of compute. You do not buy your own AWS GPUs yet).

The Layout Math: Satori (Takes HTML/React code and your custom .otf font files, and turns them into perfect SVGs so the text never overlaps the product).

The Compositor: Sharp (Node.js library). It takes the Fal.ai background, the transparent product cutout, and the Satori text layer, and flattens them into a single 4K PNG.

LAYER 6: Database & Memory (The Vault)

Where the client data and your machine learning memory live.

Primary Database & Auth: Supabase (PostgreSQL). Stores user accounts, billing IDs, and the relational data for the Brand_ID rules.

Vector Database (Style RAG): pgvector (Inside Supabase. Stores the mathematical embeddings of your "Golden Reference" magazine layouts to anchor the AI's aesthetic).

Blob Storage: Cloudflare R2 (Stores the final 4K images and custom font files. It is S3-compatible but has Zero Egress Fees, saving you tens of thousands of dollars a month in bandwidth costs).

LAYER 7: Observability & Telemetry (The Monitors)

If an enterprise client complains an ad looks wrong, you need to know exactly which agent failed.

LLM Tracing: LangSmith or Braintrust (Logs every single prompt sent to Claude/Fal.ai, the token cost, and the exact JSON response. Essential for debugging AI).

System Errors: Sentry (Catches TypeScript/Node.js crashes).

Infrastructure Metrics: Datadog (Monitors server health).

LAYER 8: Developer Experience (The Storefront)

How developers actually interact with DreamCut.

API Documentation: Mintlify (Takes your OpenAPI spec and generates a beautiful, interactive, Stripe-like documentation site with dark mode and code playgrounds).

SDKs & GenUI: React / Tailwind / Shadcn UI. Used to build your @dreamcut/ui npm package so developers can drop your <GenerativeAd /> and <BrandOnboarding /> components directly into their apps.
